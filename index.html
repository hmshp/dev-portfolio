손혜미

프론트엔드 개발자

몰랐던 사실을 알게 되는 순간을 좋아하는
프론트 개발자 손혜미입니다.

번역가로 일할 때 플레이스홀더나 익스케이프 레터가 들어간 텍스트가 나오면 어려웠지만, 유난히 재미있었어요. 이런 번역을 잘하고 싶어서 coursera에서 html과 css 강의를 들은 게 제 첫 코딩 공부였어요.

번역을 보조해 주는 툴을 사용해서 작업했는데 가끔 에러가 발생해서 작업을 멈출 수밖에 없었어요. 고객센터가 해외에 있어서 답변을 받으려면 며칠씩 걸리는데 마감 기한은 정해져 있어서 무척 난감했어요.

‘에러가 왜 나는지 이해하고 직접 고칠 수 있다면 얼마나 좋을까.’ 하는 생각을 하게 되었죠.

프로그래밍을 본격적으로 배우고 싶어서 웹 개발 국비 학원에 등록했어요.
사용자와 맞닿아 있는 작업이 더 재미있어서 주말에는 수업에서 자세히 알려주지 않는 CSS와 JavaScript 동작 원리, React를 배워서 To-do-list나 여행 일지 같이 눈에 보이는 작은 프로그램을 만들었어요.

나중에 백엔드 개발자와 협업하면서 '프론트 개발자라고 해도 어느 정도의 백엔드 지식은 있어야겠구나'를 깨닫고 node.js, express를 배워서 혼자서 풀스택 앱을 만들었어요. 그동안 전체 사이클 중 프론트라는 한 파트밖에 몰라서 답답했는데, 전체 흐름이 눈에 들어오니 속이 시원했어요.

앞으로도 속 시원한 즐거움을 느끼면서 배워나가고 싶고, 회사에서 팀원들과 함께 앱을 만들면서 프론트 개발자로서 기여하고 싶습니다.
